Analüüsiahela Üldine Töövoog (Workflow)

1. Kogumine & Puhastamine

`pcapng-analys.py` või `vahendaja.py` abil eraldatakse logidest/salvestustest toorandmed või unikaalsed IP-d.

2. Süvaanalüüs & Parsimine

Spetsiifilised skriptid (`http_traffic_analyzer.py`, `dns_andmed.py`, `tavatext-to-csv.py`) parsuvad protokolli/logi andmed.

3. Rikastamine & Teisendus

`siduja.py` (GeoIP) ning pärsitud andmetest luuakse ühtsed struktureeritud CSV-failid.

4. Visualiseerimine

Visualiseeritakse tulemused veebibrauseris vastavate HTML dashboardide abil (Drag & Drop interaktsioon).

1. Universaalne HTTP Toiming

Põhimõte: Üldise veebiliikluse (GET, POST jne) analüüs, päringute sisu ja entropia kontroll ning dünaamiline visualiseerimine.

Universal Web

http_traffic_analyzer.py

PYTHON

Tegevus: Universaalne HTTP päringute ja payloadide turvaanalüsaator. Parsib erinevaid HTTP meetodeid, kontrollib pakkimist (GZip) või struktuure (JSON, TLS) ning arvutab Shannoni entropia krüptitud või pahatahtliku sisu tuvastamiseks.

# Käivitamine:
python3 http_traffic_analyzer.py logi.txt

http_raport_to_csv.py

CONVERTER

Tegevus: Teisendab `http_traffic_analyzer.py` genereeritud TXT raporti struktureeritud CSV-failiks (semikooloniga eraldatud, UTF-8-BOM). Valmistab andmed ette universaalses vaaturis kuvamiseks.

# Käivitamine:
python3 http_raport_to_csv.py raport.txt http_analuys.csv

universaalne_vaatur.html

HTML DASHBOARD

Tegevus: Interaktiivne BI-tüüpi vaatur veebibrauserile. Võimaldab kasutajal ise valida graafikute andmeallikaid (HTTP meetod, sisu tüüp, entropia, suurus) ja graafiku tüüpe (Doughnut, Bar, Pie) ning filtreerida tabelit.

# Kasutamine:
Ava brauseris & laadi üles `http_analuys.csv`

2. IP-aadressite ja GeoIP Toiming

Põhimõte: Võrguliiklusest IP-aadresside eraldamine, filtreerimine, geograafiline rikastamine (GeoIP/ASN) ja geograafiline visualiseerimine.

IP & GeoIP Processing

pcapng-analys.py

PCAP PARSER

Tegevus: Loeb PCAP/PCAPNG faile, eraldab unikaalsed IP-aadresside paari andmed, tuvastab HTTP POST päringud ning otsib selgetekstilisi paroole.

# Käivitamine:
python3 pcapng-analys.py

vahendaja.py

IP FILTER

Tegevus: Lihtne IP-aadresside filtreerimise vahetööriist. Korjab suvalistest tekstifailidest või logidest regulaaravaldise abil välja unikaalsed IPv4 aadressid ja puhastab nimekirja.

# Käivitamine:
python3 vahendaja.py

siduja.py

GEOIP ENRICHMENT

Tegevus: Võtab puhastatud IP-aadresside nimekirja ja pärib `asn.cymru.com` API kaudu iga IP kohta riigi, internetiteenuse pakkuja (ISP) ning ASN/organisatsiooni andmed, genereerides CSV-faili.

# Käivitamine:
python3 siduja.py

graafik.html

HTML DASHBOARD

Tegevus: Visuaalne GeoIP dashboard. Kuvab `siduja.py` väljund-CSV põhjal riikide jaotust ringdiagrammina (Doughnut) ning organisatsioonide ja ASN-ide esinemissagedust tulpdiagrammina.

# Kasutamine:
Ava brauseris & laadi üles `tulemused_IP.csv`

3. DNS Päringute ja Domeenide Toiming (`dns` kaust)

Põhimõte: DNS võrgupakettide logi parsimine, rollide (Klient vs DNS-server) tuvastamine, juurdomeenide klastreerimine ja grupeeritud kuvamine.

DNS Inspection

dns_andmed.py

DNS PARSER

Tegevus: Eraldab TXT logist DNS pakettide numbrid, IP-aadressid ja pärsitud domeeninimed. Tuvastab automaatselt rolli (päring tehti kliendi poolt või vastus DNS serverilt) ja eraldab peamise juurdomeeni (nt `google.com`). Salvestab tulemused CSV-faili.

# Käivitamine (käsurea argumendiga või ilma):
python3 dns_andmed.py dns_logi.txt

dns_vaatur.html

HTML DASHBOARD

Tegevus: Interaktiivne vaatur DNS päringute analüüsiks. Võimaldab lohistada (Drag & Drop) `dns_andmed.py` poolt loodud CSV-faili, grupeerida päringuid juurdomeenide kaupa klastritesse, filtreerida IP-de põhjal ning otsida täpseid subdomeene.

# Kasutamine:
Ava brauseris & laadi üles `dns_logi.csv`

4. Muu / Tavalogi ja Plaintext Toiming (`muu` kaust)

Põhimõte: Üldiste ja struktureerimata või mitmerealiste tekstilogide (nt SMTP, SSH, süsteemilogid) teisendamine struktureeritud CSV-ks ning interaktiivne analüüs.

Plaintext & Multi-line Logs

tavatext-to-csv.py

TEXT PARSER

Tegevus: Üldotstarbeline tekstilogi teisendaja (ilma väliste sõltuvusteta nagu Pandas). Tuvastab mitmerealisi struktuure (nt SMTP ühenduse plokid, paketi numbrid, IP:Port suunad ja teated) ning tekitab puhta, Exceli-sõbraliku semikooloniga CSV-faili.

# Käivitamine:
python3 tavatext-to-csv.py logi.txt

tavatext.html

HTML DASHBOARD

Tegevus: Universaalne tekstilogide vaatur brauserile. Tuvastab automaatselt CSV-st võtmed ja väljad, toetab lohistamist (Drag & Drop), võimaldab valida grupeerimise alust (nt IP, pakett, veakood) ning tõstab esile veakoodid (nt SMTP 550 tähistused).

# Kasutamine:
Ava brauseris & laadi üles `tavalogi.csv`

Samm-sammuline Tööjuhis Kasutajale

  1. Töökausta ettevalmistamine: Tee kaustast toolbox koopia. Tõsta analüüsitav .pcapng või tekstilogi fail kopeeritud kausta.
  2. Esmane analüüs (PCAPNG puhul): Käivita kopeeritud kaustas skript pcapng-analys.py toorandmete kättesaamiseks.
  3. Failide liigutamine alamkaustadesse: Tõsta genereeritud HTTP fail http kausta, IP-d ip kausta, DNS logid dns kausta ning muud tekstilogid muu kausta.
  4. HTTP liikluse töötlus (http kaustas):
    • Käivita http_traffic_analyzer.py.
    • Käivita http_raport_to_csv.py (genereerib CSV-faili).
    • Ava brauseris universaalne_vaatur.html ja laadi CSV-fail tulemuste vaatamiseks üles.
  5. IP-aadresside töötlus (ip kaustas):
    • Käivita kõigepealt vahendaja.py.
    • Käivita seejärel siduja.py.
    • Ava brauseris graafik.html ja laadi genereeritud CSV üles graafiku vaatamiseks.
  6. DNS päringute töötlus (dns kaustas):
    • Käivita dns_andmed.py sinu_dns_logi.txt (genereerib CSV-faili).
    • Ava brauseris dns_vaatur.html ja lohista loodud CSV fail lehele.
  7. Muude tekstilogide töötlus (muu kaustas):
    • Käivita tavatext-to-csv.py logi.txt (genereerib CSV-faili).
    • Ava brauseris tavatext.html ja lohista CSV-fail vaaturisse.
